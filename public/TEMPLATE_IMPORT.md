# Panduan Format File Word untuk Import

Agar sistem dapat membaca file Word (.docx) Anda dengan benar, pastikan Anda menggunakan format teks berikut dalam dokumen Anda.

## Struktur Dokumen

Dokumen harus memiliki header/judul bagian yang jelas. Sistem akan mencari kata kunci berikut (tidak peka huruf besar/kecil):

1.  **Sumber / Penyelenggara**
    *   Kata kunci: `Sumber`, `Penyelenggara`, `Keputusan`, `Hasil`
    *   Contoh: `Sumber: LBMNU Jawa Timur 2024`

2.  **Judul / Topik Masalah**
    *   Kata kunci: `Judul`, `Topik`, `Tema`
    *   Contoh: `Judul: Hukum Asuransi Jiwa`

3.  **Pertanyaan / Deskripsi Masalah**
    *   Kata kunci: `Pertanyaan`, `Soal`, `Deskripsi Masalah`
    *   Isi: Teks pertanyaan atau deskripsi lengkap masalah.

4.  **Jawaban / Rumusan**
    *   Kata kunci: `Jawaban`, `Rumusan`, `Keputusan`
    *   Isi: Ringkasan jawaban atau hasil rumusan dalam bahasa Indonesia.

5.  **Ibarat / Referensi**
    *   Kata kunci: `Ibarat`, `Nash`, `Dasar Hukum`, `Referensi`
    *   Isi: Teks Arab dan referensi kitab.

---

## Contoh Isi File Word

Silakan copy teks di bawah ini ke file Word baru untuk mencoba:

```text
Judul:
Hukum Menggunakan WiFi Tetangga

Sumber:
Hasil Bahtsul Masail LBMNU Cabang Kencong, 12 Rabiul Awal 1445 H

Deskripsi Masalah:
Bagaimana hukumnya menggunakan WiFi tetangga tanpa izin jika sinyalnya masuk ke rumah kita?

Rumusan Jawaban:
Hukumnya haram karena termasuk ghasab (mengambil hak orang lain tanpa izin), kecuali jika ada dugaan kuat (zhan) bahwa pemiliknya merelakan (ridha).

Ibarat:
(Fathul Mu'in, Hal 120)
ﻭﻳﺤﺮﻡ ﺍﻟﺘﺼﺮﻑ ﻓﻲ ﻣﻠﻚ ﺍﻟﻐﻴﺮ ﺑﻐﻴﺮ ﺇﺫﻧﻪ
Artinya: Haram menggunakan milik orang lain tanpa izinnya.
```
